Chapter 1 A Solutions Manual

This is an unofficial solutions manual for Hastie, Tibshirani, and Wainwright (2015). I happily accept additions to the solutions through pull requests on github.

1.1 Compiling this Book

To compile this example to PDF, you need XeLaTeX. You are recommended to install TinyTeX (which includes XeLaTeX): https://yihui.org/tinytex/.

References

Hastie, Trevor, Robert Tibshirani, and Martin Wainwright. 2015. Statistical Learning with Sparsity: The Lasso and Generalizations. 1 edition. Boca Raton: Chapman and Hall/CRC.
